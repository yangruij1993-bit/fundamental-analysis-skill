# 基本面分析（/analyze）

对指定公司按3C3D5M3T 框架做完整基本面分析，生成 Markdown 文档。

## 参数

- `$ARGUMENTS` - 公司代码和（可选）侧重（如 `COST`、`600519`、`NVDA 做空`）

🛑 **STOP** — 如果参数为空，提示用户输入公司代码。

## 执行

调用 `fundamental-analysis` skill，按 SKILL.md 流程执行：

1. 解析 ticker + 判断市场（美股/A股/H股）
2. 并行拉客观数据（WebSearch + tdx MCP）
3. 按 5M 逐项分析
4. 按 3T 填写时段判断
5. 按 3D 判断股价驱动
6. LOGOS 风险清单排查
7. 得出明确结论（做多/做空/观望/观察）
8. 生成 Markdown 文档到个股目录
9. 抛出待确认的主观判断给用户

## 关键规则

- **结论必须明确**：做多/做空/观望/观察，四选一，不模棱两可。
- **仓位必须有数字**：初始 2%，上限 8%，止损位具体到价格或百分比。
- **主观判断抛给用户**：管理层诚信、商业模式简单度等不要 Claude 猜。
- **数据要有历史分位**：PB/PE 不看单点，看 5 年/10 年区间位置。
- **写作风格遵循项目 CLAUDE.md**：用词克制、短句密集、句号结尾。

## 输出位置

- 美股：`美股/个股/<sector>/<TICKER>/研究笔记/<TICKER>_基本面分析_<YYYYMMDD>.md`
- A股：`A股H股/A股/<行业>/<名称>/研究笔记/<名称>_基本面分析_<YYYYMMDD>.md`
- H股：`A股H股/H股/<行业>/<名称>/研究笔记/<名称>_基本面分析_<YYYYMMDD>.md`
- 找不到目录：`其他笔记/公司分析/`
