# fundamental-analysis · 基本面分析框架（胡猛3C3D5M3T）

> 来源：《风和投资随笔1》胡猛。完整方法论见 `framework.md`。

## 解决什么问题

看到一家公司后，不知道该问什么问题、按什么顺序问、最后怎么得出结论。这个 skill 把胡猛的 3C3D5M3T + LOGOS 方法论变成一套可执行流程：

1. **输入**：公司代码（ticker）
2. **流程**：Claude 拉客观数据 → 按 5M3T3D 逐项分析 → 填入主观判断 → 得出结论
3. **输出**：结构化的 Markdown 分析文档，含做多/做空结论、仓位建议、止损位、跟踪计划

## 使用方式

通过 `/analyze <ticker>` 调用。例如：

```
/analyze COST
/analyze 600519
/analyze NVDA
```

## 执行流程

### 步骤 1：解析参数 + 初始化

🔴 **CHECKPOINT**：
- [ ] ticker 已解析（美股如 COST，A股如 600519，港股如 0700）
- [ ] 判断市场：美股 / A股 / H股

**决定输出路径**：
- 美股 → `美股/个股/<sector>/<TICKER>/研究笔记/<TICKER>_基本面分析_<YYYYMMDD>.md`
- A股 → `A股H股/A股/<行业>/<名称>/研究笔记/<名称>_基本面分析_<YYYYMMDD>.md`
- H股 → `A股H股/H股/<行业>/<名称>/研究笔记/<名称>_基本面分析_<YYYYMMDD>.md`
- 找不到对应目录 → 放在 `其他笔记/公司分析/` 下

### 步骤 2：拉取客观数据（Claude 自动）

并行调用以下工具：

**美股**：
- WebSearch：最新 10-Q/10-K 关键数据、行业TAM、市场份额数据
- WebFetch（如需要）：直接抓公司 IR 页面或 10-K
- 关注数据：TTM 收入/利润/自由现金流、ROE、ROIC、PB、PE、毛利率、营业利润率、市场份额、行业增速

**A股**：
- tdx MCP（`eltdx_kline`、`eltdx_quote`、`eltdx_stock_profile`、`eltdx_company_profile`、`eltdx_stock_topics`）：行情、F10、题材
- WebSearch：补充行业TAM、卖方研究

**H股**：
- WebSearch：HKEXnews 公告、年报
- 关注数据：和美股类似

**数据要求**：
- 财务数据至少 5 年历史，看趋势不是看单点
- 估值数据看历史区间分位（当前 PB/PE 在过去 5 年/10 年的什么位置）
- 同行数据至少 3 家可比公司

### 步骤 3：5M 分析（按框架逐项填写，标注多空偏向）

**严格按 `framework.md` 第三章的问题清单逐项回答**。每个 M 写 2-3 段现状+趋势+关键变量，**并在每个 M 末尾标注多空偏向**。

**M1 目标市场（TAM）**——**做空偏向（做空基础）**：
- 行业多大？是否触及天花板？
- 增速来源（渗透率/消费升级/技术/一次性）？
- 需求是永久的还是一次性？
- 做多时 M1 只要"不致命"即可；做空时 M1 到顶/萎缩/需求一次性是首要信号。

**M2 市场份额**——**做多核心 + 结构做空信号**：
- 当前份额？趋势？来源（产品力/价格/渠道/品牌）？
- 同行表现（全行业差=周期，同行好它差=结构）？
- 份额提升的代价（烧钱 vs 产品力驱动）？
- 边际变好=做多信号；同行好它差=结构做空信号。

**M3 利润率（OPM）与现金流**——**做多核心 + 周期做空信号**：
- 毛利率/营业利润率趋势？驱动力？
- 赚的是现金还是应收款？**自由现金流才是真现金**。
- 营销/研发/管理费用趋势？营销飙升是预警。
- 利润率是否在历史峰值？是否周期顶部？
- 边际扩张 + 硬核现金=做多；峰值 + 行业向下=周期做空。

**M4 商业模式（是否简单）**——**做空偏向（做空基础）**：
- 一句话能否说清怎么赚钱？
- 规模效应在哪里？
- 越做越轻松还是越做越累？
- 是否依赖单一客户/供应商/政策/管理层？
- 做多时 M4 是筛选条件；做空时 M4 变复杂/跨界多元化/持续烧钱是预警。

**M5 管理团队与治理**——**做多必要 + 结构做空触发**：
- 管理层和中小股东利益是否一致？
- 治理透明度？信息披露质量？
- 企业家是否还在一线？是否陷入"跨界指点"的认知陷阱？
- 资本配置能力？
- 做多时 M5 诚信一致是必要条件；做空时 M5 恶化是结构做空触发点。**高 ROE + 优秀管理层是做空大敌**。

### 步骤 4：3T 时间框架分析（标注多空主战场）

对每个 T 给出"好/中/差"的判断 + 一句话依据：

- **T1（0-3月）**：下一份季报预期？短期catalyst？情绪/估值驱动？
- **T2（3-18月）**：盈利趋势？产品周期/库存周期/产能周期位置？**（研究核心战场）**
- **T3（18月+）**：行业格局变化？管理层更替？模式颠覆？

**多空组合判断**：
- **T1 差 + T2 好 + 估值低** = **左侧做多主战场**（胡猛偏好）。
- T1 好 + T2 好 = 右侧做多（不追求，"错失股票不是错"）。
- T1 好 + T2 差 + 估值高 + catalyst = 宴席式做空（左侧）。
- **T1 差 + T2 差 + 顺势** = **踩烂式做空主战场**（确定性高）。

### 步骤 5：3D 股价驱动判断（多空两义）

- **D1 ROE**：长期ROE水平？是否≥10%（胡猛标准）？是否硬核ROE？**高 ROE 是做多根基 / 做空大敌**——做空尽量避开高 ROE 公司。
- **D2 价值外延变化**：当前是结构变化还是周期变化？方向（变好/变差）？**变好是做多驱动；变差是做空基础**。
- **D3 情绪/估值**：当前PB/PE在历史区间位置？拥挤度？市场情绪？**底部是做多入场时机；顶部 + catalyst 才能空估值**。

### 步骤 6：LOGOS 风险清单排查

**对照 `framework.md` 第七章的44个核心问题**逐项排查，列出：
- 触发的风险点（如有）
- 关键盲点（Claude 或数据无法判断的）
- 需要用户确认的主观判断

**做空追加排查**（仅在做空结论时）：44 个问题中的 38-44（拥挤度、X factor、被并购价值等）。

### 步骤 7：综合结论（多空对称）

**必须得出明确结论**，不能模棱两可：

| 情况 | 结论 |
|------|------|
| M2/M3/M5 边际变好 + T2 好 + 估值偏低 + D1 硬核 ROE | **做多（左侧）** |
| T1 差 + T2 好 + 估值历史低位 | **做多（主战场）** |
| M1/TAM 到顶 + M3 周期向下 + T1 差 + T2 差 + 估值有 catalyst | **做空（踩烂式，周期做空）** |
| M2 份额丢失 + 同行好它差 + M4/M5 恶化 | **做空（结构做空，需谨慎）** |
| T1 好 + T2 差 + 估值顶部 + 明确 catalyst | **做空（宴席式，难度高）** |
| 部分好部分差 + 估值不便宜 | **观望** |
| 数据不足或主观判断不明确 | **观察（列待确认清单）** |

**仓位建议**（默认）：
- 初始仓位 2%
- 依据下行空间加仓，单只最高 8%
- 明确止损位（多头一般 -15% 到 -20%，视波动率）

**做空额外的铁律检查**：
- 不做空骗子（不和坏人纠缠）。
- 做空曾经伟大的企业（高 ROE 但结构性衰落）。
- 做空最大对手是优秀管理层——若 M5 优秀 + D1 高 ROE，做空需要极高确信。
- 估值做空必须有 catalyst（否则恒瑞可以一直 60 倍）。

## 写作风格（遵循项目 CLAUDE.md）

- **用词克制**：不用"令人震惊""史无前例""显著""强劲"

## 方法论核心（一页速查）

> 详细请读 `framework.md`。

**3C 心态**（不是信号）：Cycle周期谦卑 / Change关注变化 / Certainty确定性定仓。
**3D 驱动**：
- D1 ROE（高=做多根基 / 做空大敌）
- D2 价值外延变化（变好=做多驱动 / 变差=做空基础）
- D3 情绪估值（底部=做多时机 / 顶部+catalyst=做空时机）

**5M 多空偏向**：
- **做多核心**：M2 份额 + M3 利润率 + M5 团队（边际变好）
- **做空基础**：M1 TAM 到顶 + M4 模式危机
- **周期做空信号**：M1/M3 行业波动
- **结构做空信号**：M2 个股波动（背后 M4/M5 恶化）

**3T 多空主战场**：
- **做多主战场**：T1 差 + T2 好 + 估值低（左侧，胡猛偏好）
- **做空主战场**：T1 差 + T2 差（踩烂式，顺势）
- 宴席式做空（T1 好 + T2 差 + 估值顶 + catalyst）难度高

**LOGOS**：100问风险清单。

**核心原则**：
1. 要观测、要推测、不要预测。
2. Think Small，自下而上。
3. 投资变化（好价钱买变好的企业）。
4. 投资+怀疑。
5. 多中空大（做多中型企业，做空恐龙级企业）。
6. 该大的大、该小的小（单仓≤8%）。
7. 错失股票不是错、买错股票才是错。

**做空铁律**：
- 不做空骗子，做空"曾经伟大"
- 做空行业周期 > 做空结构个股
- 做空最大对手是优秀管理层（高 ROE + 优秀 M5 = 避空）
- 估值做空必须有 catalyst
